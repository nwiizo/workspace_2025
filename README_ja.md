# 2025年の目標 🎯

### 目標1：体重69kg達成とキックボクシング試合出場 🥊

#### 栄養の基礎と体重管理 📊
炭水化物、タンパク質、脂肪の3つがエネルギーになって、これが余ると太るわけです。7200kcalで1kg太ります。

たいてい炭水化物は取りすぎなんで、それを減らします。全体として基礎代謝以上消費カロリー未満にします。成人男性なら1500-2000kcalくらい。

筋肉とかを作るのはタンパク質なんでこれは減らさない。脂肪と炭水化物を減らす。これだけ。

1. **健康的な食事計画の実施:** 🥗
   - [ ] 1日1800kcalを目標とした具体的な食事プラン作成
   - [ ] 1食あたり野菜350g以上の摂取
   - [ ] 週次での食事記録とレビュー
   - [ ] 間食は果物またはノンカロリー飲料のみに制限

2. **キックボクシングトレーニング:** 🥊
   - [ ] キックボクシングジムに入会し週5回通う
   - [ ] 基本コンビネーションの毎日練習（30分）
   - [ ] 月2回のスパーリング参加
   - [ ] アマチュア試合デビューの具体的なスケジュール設定

3. **健康管理:** ⚕️
   - [ ] 毎週月曜日の体重・体脂肪率記録
   - [ ] 3ヶ月ごとの健康診断データ比較
   - [ ] 就寝時間の管理（23:30 - 06:30）
