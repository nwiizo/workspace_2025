# Punchline Review - パンチライン・キーメッセージレビュー

このコマンドは、指定されたコンテンツを「記憶に残るフレーズ」「引用可能性」「SNSシェア適性」の観点から分析し、キーメッセージの強化提案を提供します。

## 使用方法

```
/punchline-review [ファイルパス]
```

例:
```
/punchline-review blog.md
```

## 評価観点

以下の観点で評価します：

1. **粘着性（Stickiness）** - フレーズが記憶に残る度合い
2. **引用可能性（Quotability）** - ビジネス・日常で引用されやすさ
3. **SNSシェア適性（Virality）** - SNSで拡散されやすさ
4. **核心メッセージの明確さ** - 各章のキーメッセージが明確か
5. **具体性と抽象性のバランス** - 概念が端的な一言に落ちているか

## 評価手順

### Step 1: コンテンツを読む

指定されたファイルを全文読み、以下をメモする：
- 全体のキーメッセージ
- 各章の核心メッセージ
- 記憶に残るフレーズの候補

### Step 2: パンチライン発見と分析

#### 強力なパンチライン TOP 10の特定
各フレーズについて：
- 行番号
- 引用（そのまま）
- なぜ記憶に残るか（構造・リズム・意外性）
- 粘着性スコア（0.0-10.0）

**粘着性スコアの基準:**
- 10.0-9.0: 極めて強力（数年後も記憶に残る）
- 8.9-8.0: 非常に強力（読後1週間は確実に記憶に残る）
- 7.9-7.0: 強力（読後1日は記憶に残る）
- 6.9-6.0: やや弱い（読後数時間で忘れる可能性）
- 5.9以下: 弱い（即座に忘れる）

#### パンチラインの3要素分析
優れたパンチラインは以下の要素を持つ：

1. **シンプルさ** - 短く、分かりやすい
2. **意外性** - 予想外の切り口や対比
3. **真実性** - 読者の体験と共鳴する

### Step 3: 強化提案の作成

#### 潜在的パンチラインの強化
現在の表現を、より記憶に残る形にリライトします。

**リライトの技法:**
- **対比の強化**: 「AではなくB」「AよりB」
- **数式化**: 「A < B」「A + B = C」
- **問いの形**: 「〜か？」で終わらせる
- **断定の強化**: 曖昧な表現を削る
- **リズムの調整**: 7-5調、5-7-5などのリズム
- **比喩の導入**: 抽象概念を具体的イメージに

### Step 4: SNSシェア適性分析

#### 140-280字シェア可能フレーズの抽出
以下の基準で評価：
- 単独で意味が通じるか
- 文脈なしでも理解できるか
- 共感を呼ぶか
- リツイート/シェアされやすいか

### Step 5: 各章の核心メッセージ評価

各章について：
- 核心メッセージは何か（1文で）
- 明確度スコア（1-5）
- 改善提案（必要な場合）

## 評価の基本姿勢

1. **発掘する** - 既に存在する強力なフレーズを見逃さない
2. **磨く** - 潜在力のあるフレーズをより記憶に残る形に
3. **具体的に示す** - 抽象的指摘ではなく、具体的なリライト案を提供
4. **読者目線** - 読者の記憶に残るかを最優先
5. **実用性重視** - SNS・会議・日常で使えるフレーズか

---

## 重要な注意事項

- パンチラインは「奇をてらう」のではなく、「本質を端的に表現する」もの
- 著者の声（voice）を壊さず、磨き上げる
- 全てのフレーズを変える必要はない。強化すべき箇所を厳選する
- 記憶に残るのは、シンプルで力強い言葉
